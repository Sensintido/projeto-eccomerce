#  TechStore | E-commerce Hardware & Games

> **Status do Projeto:**  Em Desenvolvimento /  Finalizado

##  Sobre o Projeto

A **TechStore** é uma plataforma de e-commerce completa voltada para o público entusiasta de hardware e tecnologia. O design e a experiência do usuário foram inspirados na gigante **KaBuM!**, focando em uma listagem de produtos eficiente, filtros avançados e um checkout fluido.

Este projeto foi desenvolvido para colocar em prática conceitos de **arquitetura de software, gerenciamento de estado e integração com meios de pagamento**.

---

##  Funcionalidades Principais

* **Busca Inteligente:** Encontre produtos por nome, categoria ou marca.
* **Filtros Dinâmicos:** Filtre por preço, disponibilidade e especificações técnicas.
* **Carrinho em Tempo Real:** Adicione e remova itens com atualização instantânea de valores.
* **Área do Cliente:** Histórico de pedidos e rastreamento.
* **Modo Gamer (Dark Mode):** Interface adaptável para melhor conforto visual.
* **Checkout Responsivo:** Otimizado para desktop e dispositivos móveis.

##  Como Executar o Projeto

### Pré-requisitos

Antes de começar, você precisará ter instalado em sua máquina:

* [Git](https://git-scm.com)
* [Node.js](https://nodejs.org/en/)
* Um gerenciador de pacotes como **npm** ou **yarn**.

### Passo a passo

1. **Clone o repositório:**
```bash
git clone https://github.com/seu-usuario/nome-do-projeto.git

```


2. **Instale as dependências:**
```bash
cd nome-do-projeto
npm install

```


3. **Configure as variáveis de ambiente:**
Crie um arquivo `.env` na raiz do projeto e adicione as chaves necessárias (exemplo no arquivo `.env.example`).
4. **Inicie o servidor de desenvolvimento:**
```bash
npm run dev

```


Acesse `http://localhost:3000` no seu navegador.
